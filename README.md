# 🩸 Blood Donor Finder (Streamlit App)

## 📌 Description
A simple web application built using Streamlit to find and register blood donors based on blood group and city.

## 🚀 Features
- Search donors by blood group and city
- Add new donors
- Simple and user-friendly interface

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas

## 📂 Project Structure
app.py        -> Main UI
model.py      -> Data handling
utils.py      -> Helper functions
donors.csv    -> Data storage
